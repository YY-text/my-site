📖 参考示例

示例1：好的和坏的命名
# 好的命名: 有描述性，使用下划线或驼峰 # 坏的命名: 只有a,b,c或拼音，难以理解 # 好的命名 student_name = "小明" course_score = 95 max_temperature = 38.5 # 可读性对比 # 这段代码在说什么？ a = 95 b = 88 c = a + b # 这段呢？ chinese_score = 95 math_score = 88 total_score = chinese_score + math_score print(f"总分为: {total_score}")
总分为: 183
示例2：变量命名规范
# 命名规则: # 1. 必须以字母或下划线开头 # 2. 只能包含字母、数字和下划线 # 3. 不能使用Python关键字（如if, for, while等） # 4. 区分大小写: Name 和 name 是两个不同变量 # 5. 推荐: 使用小写+下划线 (snake_case)，见名知意 # 推荐写法 user_age = 18 user_name = "小明" user_city = "北京" print(f"用户信息: {user_name}, {user_age}岁, {user_city}")
用户信息: 小明, 18岁, 北京
