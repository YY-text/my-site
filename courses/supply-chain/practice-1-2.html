💻 供应链流程 - Python实操

第1章 供应链概述

供应链流程练习
供应链流程实操练习：计划：需求预测、资源规划、采购：供应商选择、订单下达、生产：制造、装配、质量控制、仓储：入库、存储、出库、运输：配送、物流跟踪、销售：分销、零售、客户服务、逆向物流：退货、回收
本节知识点：
  • 计划：需求预测、资源规划
  • 采购：供应商选择、订单下达
  • 生产：制造、装配、质量控制
  • 仓储：入库、存储、出库
  • 运输：配送、物流跟踪
  • 销售：分销、零售、客户服务
  • 逆向物流：退货、回收

📖 参考示例

示例1：查看Python版本
# import 语句用于导入Python的内置模块 # sys 模块提供了很多与系统交互的功能 import sys print("=" * 40) print(" Python版本信息") print("=" * 40) print(f"主版本号: {sys.version_info.major}") print(f"次版本号: {sys.version_info.minor}") print(f"微版本号: {sys.version_info.micro}") print(f"发行级别: {sys.version_info.releaselevel}") print(f"完整版本字符串: {sys.version}") print("=" * 40)
======================================== Python版本信息 ======================================== 主版本号: 3 次版本号: 11 微版本号: 0 发行级别: final 完整版本字符串: 3.11.0 ... ========================================
示例2：查看系统信息
import sys import platform print("=" * 40) print(" 系统信息") print("=" * 40) print(f"操作系统: {platform.system()}") print(f"详细版本: {platform.platform()}") print(f"架构: {platform.architecture()[0]}") print(f"处理器: {platform.processor()}") print(f"Python路径: {sys.executable}") print(f"默认编码: {sys.getdefaultencoding()}") print("=" * 40)
======================================== 系统信息 ======================================== 操作系统: Windows 详细版本: Windows-10-10.0.19041 架构: 64bit 处理器: Intel64... Python路径: C:\Python311\python.exe 默认编码: utf-8 ========================================
