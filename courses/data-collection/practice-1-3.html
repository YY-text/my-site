📖 参考示例

示例1：交互式输入
# input() 函数让用户在命令行输入内容 # input返回的永远是字符串类型 name = input("请输入你的名字: ") print(f"你好, {name}! 欢迎来到Python世界!") print(f"{name}, 今天想学习什么呢？")
请输入你的名字: 小明 你好, 小明! 欢迎来到Python世界! 小明, 今天想学习什么呢？
示例2：多次输入收集信息
# 可以多次调用input收集不同信息 print("=" * 30) print(" 个人信息录入") print("=" * 30) name = input("请输入你的姓名: ") age = input("请输入你的年龄: ") city = input("请输入所在城市: ") print("\n" + "=" * 30) print(f"姓名: {name}") print(f"年龄: {age}岁") print(f"城市: {city}") print(f"{name}, 欢迎你！")
============================== 个人信息录入 ============================== 请输入你的姓名: 小明 请输入你的年龄: 18 请输入所在城市: 北京 ============================== 姓名: 小明 年龄: 18岁 城市: 北京 小明, 欢迎你！
示例3：简单加法计算器
# input得到的是字符串，需用int()或float()转成数字 print("=" * 30) print(" 简易加法计算器") print("=" * 30) # 获取用户输入 num1 = input("请输入第一个数字: ") num2 = input("请输入第二个数字: ") # 转换为整数 a = int(num1) b = int(num2) # 计算并输出 result = a + b print(f"{a} + {b} = {result}") print("计算完成！")
============================== 简易加法计算器 ============================== 请输入第一个数字: 15 请输入第二个数字: 25 15 + 25 = 40 计算完成！
