📖 参考示例

示例1：字符串的基本操作
# 字符串是用引号包裹的字符序列 # 单引号、双引号、三引号都可以 s = "Hello, Python!" print(f"原字符串: {s}") print(f"长度: {len(s)}") print(f"大写: {s.upper()}") print(f"小写: {s.lower()}") print(f"首字母大写: {s.title()}") print(f"是否以Hello开头: {s.startswith("Hello")}") print(f"是否包含Python: {"Python" in s}") print(f"替换: {s.replace("Python", "世界")}")
原字符串: Hello, Python! 长度: 14 大写: HELLO, PYTHON! 小写: hello, python! 首字母大写: Hello, Python! 是否以Hello开头: True 是否包含Python: True 替换: Hello, 世界!
示例2：字符串拼接和重复
# 用 + 拼接字符串 # 用 * 重复字符串 part1 = "学习" part2 = "Python" full = part1 + part2 print(f"拼接: {full}") # 打印漂亮的分隔符 print("=" * 40) print(" " + full + " 很有趣! ") print("=" * 40) # 重复打印 print("加油! " * 5)
拼接: 学习Python ======================================== 学习Python 很有趣! ======================================== 加油! 加油! 加油! 加油! 加油!
示例3：字符串索引和切片
# 字符串的每个字符都有位置编号（索引从0开始） # s[位置] 取出单个字符 # s[开始:结束] 取出一段（切片） s = "ABCDEFG" print(f"原字符串: {s}") print(f"第1个字符: {s[0]}") print(f"第3个字符: {s[2]}") print(f"最后一个: {s[-1]}") print(f"前3个字符: {s[:3]}") print(f"第3到第5个: {s[2:5]}") print(f"后3个字符: {s[-3:]}")
原字符串: ABCDEFG 第1个字符: A 第3个字符: C 最后一个: G 前3个字符: ABC 第3到第5个: CDE 后3个字符: EFG
