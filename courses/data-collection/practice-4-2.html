📖 参考示例

示例1：浮点数计算
# 浮点数就是带小数点的数字 # 注意：浮点数计算可能有精度问题 price = 19.9 quantity = 3 total = price * quantity print(f"单价: {price}元") print(f"数量: {quantity}件") print(f"总价: {total}") print(f"格式化显示: {total:.2f}元") # 精度问题示例: 0.1 + 0.2 print(f"\n0.1 + 0.2 = {0.1 + 0.2}") print("这是浮点数的存储特性，不是bug!")
单价: 19.9元 数量: 3件 总价: 59.699999999999996 格式化显示: 59.70元 0.1 + 0.2 = 0.30000000000000004 这是浮点数的存储特性，不是bug!
示例2：round函数
# round() 函数可以四舍五入 # 语法: round(数字, 保留小数位数) value = 3.1415926 print(f"原值: {value}") print(f"保留0位: {round(value)}") print(f"保留1位: {round(value, 1)}") print(f"保留2位: {round(value, 2)}") print(f"保留4位: {round(value, 4)}")
原值: 3.1415926 保留0位: 3 保留1位: 3.1 保留2位: 3.14 保留4位: 3.1416
