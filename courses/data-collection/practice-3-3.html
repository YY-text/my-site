📖 参考示例

示例1：多种赋值方式
# 1. 普通赋值 x = 10 print(f"普通赋值: x = {x}") # 2. 链式赋值（多个变量设为同一个值） a = b = c = 5 print(f"链式赋值: a={a}, b={b}, c={c}") # 3. 多元赋值（一次给多个变量赋值） name, age, city = "小明", 18, "北京" print(f"多元赋值: {name}, {age}, {city}") # 4. 增量赋值 count = 0 count += 1 # 相当于 count = count + 1 count += 1 count += 1 print(f"增量赋值: count = {count}")
普通赋值: x = 10 链式赋值: a=5, b=5, c=5 多元赋值: 小明, 18, 北京 增量赋值: count = 3
示例2：增量赋值运算符
# 增量赋值有: +=, -=, *=, /=, %=, **=, //= score = 10 print(f"初始分: {score}") score += 5 # 加5 print(f"+5: {score}") score -= 3 # 减3 print(f"-3: {score}") score *= 2 # 乘2 print(f"x2: {score}") score /= 3 # 除3 print(f"/3: {score}")
初始分: 10 +5: 15 -3: 12 x2: 24 /3: 8.0
