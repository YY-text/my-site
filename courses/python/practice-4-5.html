💻 类型转换 - Python实操

第4章 基础数据类型

类型转换练习
类型转换实操练习：int()：转换为整数、float()：转换为浮点数、str()：转换为字符串、bool()：转换为布尔值、chr()：ASCII码转字符、ord()：字符转ASCII码、转换时注意数据丢失
本节知识点：
  • int()：转换为整数
  • float()：转换为浮点数
  • str()：转换为字符串
  • bool()：转换为布尔值
  • chr()：ASCII码转字符
  • ord()：字符转ASCII码
  • 转换时注意数据丢失

📖 参考示例

示例1：常用类型转换
# 类型转换函数: int(), float(), str(), bool() # 字符串转数字 num_str = "123" num_int = int(num_str) print(f"字符串 \"{num_str}\" 转为整数: {num_int}") print(f"类型: {type(num_str)} -> {type(num_int)}") # 数字转字符串 age = 18 age_str = str(age) print(f"数字 {age} 转为字符串: \"{age_str}\"") # 浮点转整数（截断小数部分） pi = 3.14159 print(f"{pi} 转成整数: {int(pi)} (不是四舍五入，是截断)") # 整数转浮点数 print(f"5 转成浮点数: {float(5)}")
字符串 "123" 转为整数: 123 类型: -> 数字 18 转为字符串: "18" 3.14159 转成整数: 3 (不是四舍五入，是截断) 5 转成浮点数: 5.0
示例2：实际应用
# 从input得到的是字符串，需要转成数字才能计算 print("=" * 30) print(" 简单的购物计算") print("=" * 30) price_str = input("请输入商品价格: ") qty_str = input("请输入购买数量: ") # 转成浮点数和整数 price = float(price_str) qty = int(qty_str) total = price * qty print(f"商品: {price}元 x {qty}件 = {total:.2f}元")
============================== 简单的购物计算 ============================== 请输入商品价格: 19.99 请输入购买数量: 3 商品: 19.99元 x 3件 = 59.97元
