💻 语言简介 - Python实操

第1章 Python入门认知

语言简介练习
语言简介实操练习：Python由Guido van Rossum于1991年创建、名字来源于Monty Python喜剧团、是一种解释型、面向对象的高级编程语言、设计理念：优雅、明确、简单、应用领域：Web开发、数据分析、人工智能、自动化等、主要优势：语法简洁、库丰富、社区活跃、开发效率高、主流版本：Python 3.x（Python 2已于2020年停止维护）
本节知识点：
  • Python由Guido van Rossum于1991年创建
  • 名字来源于Monty Python喜剧团
  • 是一种解释型、面向对象的高级编程语言
  • 设计理念：优雅、明确、简单
  • 应用领域：Web开发、数据分析、人工智能、自动化等
  • 主要优势：语法简洁、库丰富、社区活跃、开发效率高
  • 主流版本：Python 3.x（Python 2已于2020年停止维护）

📖 参考示例

示例1：最基础的Hello程序
# Hello World 是学习任何编程语言的第一个程序 # print() 函数用于在屏幕上输出内容 # 括号中的字符串用双引号 "" 或 单引号 '' 包裹 print("Hello, World!") print("你好，Python!")
Hello, World! 你好，Python!
示例2：多行输出与装饰
# 可以用多个print语句打印多行内容 # 使用 "=" * 数字 可以重复打印字符，画分割线 print("=" * 50) print(" 欢迎来到Python世界") print("=" * 50) print("Python是一个简单易学、功能强大的语言") print("让我们一起开始编程之旅！") print("=" * 50)
================================================== 欢迎来到Python世界 ================================================== Python是一个简单易学、功能强大的语言 让我们一起开始编程之旅！ ==================================================
示例3：特殊字符的使用
# 特殊转义字符： # \n → 换行 # \t → 制表符(Tab，用于对齐) # \\ → 反斜杠本身 print("第一行\n第二行\n第三行") print() print("姓名\t年龄\t城市") print("小明\t18\t北京") print("小红\t20\t上海") print("小张\t19\t广州")
第一行 第二行 第三行 姓名 年龄 城市 小明 18 北京 小红 20 上海 小张 19 广州
