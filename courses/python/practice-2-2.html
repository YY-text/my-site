💻 格式输出 - Python实操

第2章 基础输出与注释

格式输出练习
格式输出实操练习：f-string（推荐）：f"Hello {name}"、%格式化："Hello %s" % name、format()方法："Hello {}".format(name)、数字格式化：{:.2f}保留两位小数、对齐：{:<10}左对齐，{:>10}右对齐、千分位：{:,}自动添加千分位分隔符、百分号：{:.1%}显示为百分比格式
本节知识点：
  • f-string（推荐）：f"Hello {name}"
  • %格式化："Hello %s" % name
  • format()方法："Hello {}".format(name)
  • 数字格式化：{:.2f}保留两位小数
  • 对齐：{:<10}左对齐，{:>10}右对齐
  • 千分位：{:,}自动添加千分位分隔符
  • 百分号：{:.1%}显示为百分比格式

📖 参考示例

示例1：f-string格式化
# f-string是Python 3.6+推荐的格式化方式 # 在字符串前加f，然后用{变量名}插入变量值 name = "小明" age = 18 score = 95.5 print(f"姓名: {name}") print(f"年龄: {age}岁") print(f"成绩: {score}分") print(f"{name}今年{age}岁，考试成绩{score}分")
姓名: 小明 年龄: 18岁 成绩: 95.5分 小明今年18岁，考试成绩95.5分
示例2：数字格式化
# :.2f 表示保留2位小数 # :.0f 表示不保留小数 # :, 表示千分位分隔符 price = 19.99 quantity = 3 total = price * quantity print(f"单价: {price:.2f}元") print(f"数量: {quantity}") print(f"总价: {total:.2f}元") print(f"精确总价: {total}")
单价: 19.99元 数量: 3 总价: 59.97元 精确总价: 59.97000000000001
