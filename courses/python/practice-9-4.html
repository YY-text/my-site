💻 字典遍历 - Python实操

第9章 字典使用

字典遍历练习
字典遍历实操练习：for key in dict遍历键、for value in dict.values()遍历值、for k, v in dict.items()遍历键值对、items()返回键值对元组、可同时遍历索引和元素、字典推导式快速创建、字典的乱序特性（Python3.7+有序）
本节知识点：
  • for key in dict遍历键
  • for value in dict.values()遍历值
  • for k, v in dict.items()遍历键值对
  • items()返回键值对元组
  • 可同时遍历索引和元素
  • 字典推导式快速创建
  • 字典的乱序特性（Python3.7+有序）

📖 参考示例

示例1：基础用法
# 字典遍历 - 基础练习 # 知识点: for key in dict遍历键、for value in dict.values()遍历值、for k, v in dict.items()遍历键值对、items()返回键值对元组、可同时遍历索引和元素、字典推导式快速创建、字典的乱序特性（Python3.7+有序） # 这里是示例代码 print("=" * 30) print(" 字典遍历 练习") print("=" * 30) # 简单测试 value = 10 print(f"测试值: {value}") print(f"加倍: {value * 2}") print(f"平方: {value ** 2}") print("练习完成!")
============================== 字典遍历 练习 ============================== 测试值: 10 加倍: 20 平方: 100 练习完成!
示例2：实际应用
# 字典遍历 - 实际应用 # 根据本节知识点编写更复杂的代码 data = [1, 2, 3, 4, 5] print("=" * 30) print(" 数据处理示例") print("=" * 30) # 处理数据 for i, v in enumerate(data): print(f"第{i+1}个: {v}") print("=" * 30) print(f"总计: {sum(data)}") print(f"个数: {len(data)}") print(f"平均: {sum(data)/len(data):.1f}")
============================== 数据处理示例 ============================== 第1个: 1 第2个: 2 第3个: 3 第4个: 4 第5个: 5 ============================== 总计: 15 个数: 5 平均: 3.0
