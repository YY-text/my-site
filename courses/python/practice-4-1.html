💻 整数类型 - Python实操

第4章 基础数据类型

整数类型练习
整数类型实操练习：int：整数类型，如1、100、-5、Python3整数没有大小限制、支持多种进制：十进制0b二进制、0o八进制、0x十六进制、bin()、oct()、hex()转换进制、数学运算：+、-、*、/、//、%、幂运算：2**10表示2的10次方、整数除法：//向下取整
本节知识点：
  • int：整数类型，如1、100、-5
  • Python3整数没有大小限制
  • 支持多种进制：十进制0b二进制、0o八进制、0x十六进制
  • bin()、oct()、hex()转换进制
  • 数学运算：+、-、*、/、//、%
  • 幂运算：2**10表示2的10次方
  • 整数除法：//向下取整

📖 参考示例

示例1：整数运算
# 整数是不带小数点的数字 # 支持所有数学运算 a, b = 15, 4 print(f"{a} + {b} = {a + b}") print(f"{a} - {b} = {a - b}") print(f"{a} * {b} = {a * b}") print(f"{a} / {b} = {a / b} (结果是浮点数)") print(f"{a} // {b} = {a // b} (整除，只取整数部分)") print(f"{a} % {b} = {a % b} (取余，得到余数)") print(f"{a} ** {b} = {a ** b} (a的b次方)")
15 + 4 = 19 15 - 4 = 11 15 * 4 = 60 15 / 4 = 3.75 (结果是浮点数) 15 // 4 = 3 (整除，只取整数部分) 15 % 4 = 3 (取余，得到余数) 15 ** 4 = 50625 (a的b次方)
示例2：数字大了也不怕
# Python的整数可以非常大（不像其他语言有溢出问题） big = 2 ** 100 print(f"2的100次方 = {big}") print(f"这个数的长度: {len(str(big))} 位") # 可以用下划线让大数字更易读 population = 1_400_000_000 # 14亿 print(f"人口数: {population}人")
2的100次方 = 1267650600228229401496703205376 这个数的长度: 31 位 人口数: 1400000000人
