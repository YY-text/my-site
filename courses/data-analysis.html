第1章 数据分析概述

1.1 数据分析流程

通俗讲解

数据分析就像侦探破案，先要明确要解决什么问题，然后收集线索，整理线索，分析线索，最后得出结论。每一步都很重要，不能跳过。

核心知识点
  • 问题定义：明确分析目标和需求
  • 数据收集：获取相关数据
  • 数据清洗：处理缺失值、异常值
  • 数据分析：运用统计和机器学习方法
  • 结果呈现：可视化展示和报告
示例：电商用户行为分析流程 
1. 问题定义：
   - 如何提高用户转化率？
   - 哪些因素影响购买决策？

2. 数据收集：
   - 用户浏览记录
   - 购买历史
   - 用户画像数据

3. 数据清洗：
   - 处理缺失数据
   - 去除异常值
   - 格式统一

4. 数据分析：
   - 用户行为路径分析
   - 转化率漏斗分析
   - A/B测试

5. 结果呈现：
   - 可视化图表
   - 分析报告
   - 建议方案

本节练习

1. 数据分析的第一步是？

2. 数据清洗的目的是？

3. 以下哪个不是数据分析流程？

4. 好的分析问题应该是？

5. 结果呈现的主要方式是？

6. 数据收集时应注意？

7. 探索性数据分析(EDA)的目的是？

8. 以下哪个说法正确？

9. 数据分析报告应该？

10. 关于数据分析，正确的理解是？

1.2 分析方法分类

通俗讲解

分析方法就像看病的方法，有的是看过去的病历(描述性)，有的是诊断现在的问题(诊断性)，有的是预测未来(预测性)，有的是开药方(规范性)。

核心知识点
  • 描述性分析：发生了什么？
  • 诊断性分析：为什么发生？
  • 预测性分析：将要发生什么？
  • 规范性分析：应该怎么做？

1.3 常用工具介绍

通俗讲解

数据分析工具就像工具箱，有的简单易用(Excel)，有的功能强大(Python)，有的适合可视化(Tableau)，根据需求选择合适的工具。

核心知识点
  • Excel：适合入门，快速分析
  • Python：灵活强大，适合编程
  • R：统计分析功能强
  • Tableau：可视化效果好

1.4 商业应用场景

通俗讲解

数据分析在商业中应用广泛，就像给企业做体检，找出问题，预测未来，帮助企业做出更好的决策。

核心知识点
  • 用户画像与精准营销
  • 销售预测与库存管理
  • 风险识别与控制
  • 运营优化与效率提升

第2章 描述性统计

2.1 集中趋势

均值、中位数、众数描述数据的中心位置。

2.2 离散程度

方差、标准差、极差描述数据的分散程度。

2.3 分布形态

偏度、峰度描述数据分布的形状。

2.4 统计表图

用表格和图表直观展示数据特征。

第3章 数据可视化

3.1 可视化原则

简洁、准确、有效是好的可视化的关键。

3.2 常用图表

柱状图、折线图、散点图、饼图等各有用途。

3.3 配色设计

合理的配色让图表更清晰美观。

3.4 交互图表

交互式图表让用户可以探索数据细节。

第4章 相关性分析

4.1 相关系数

皮尔逊相关系数衡量线性相关程度。

4.2 散点图分析

散点图直观展示两个变量的关系。

4.3 协方差矩阵

协方差矩阵展示多个变量间的关系。

4.4 因果与相关

相关不等于因果，需要谨慎判断。

第5章 假设检验

5.1 基本概念

原假设、备择假设、p值是假设检验的核心。

5.2 t检验

t检验用于比较两组均值是否有显著差异。

5.3 卡方检验

卡方检验用于检验分类变量的独立性。

5.4 方差分析

方差分析用于比较多组均值。

第6章 回归分析

6.1 线性回归

用直线拟合自变量和因变量的关系。

6.2 多元回归

用多个自变量预测因变量。

6.3 逻辑回归

用于分类问题，预测概率。

6.4 模型评估

R方、MSE等指标评估模型好坏。

第7章 聚类分析

7.1 K-means聚类

K-means将数据分成K个簇。

7.2 层次聚类

层次聚类建立树状的聚类结构。

7.3 DBSCAN

DBSCAN基于密度聚类，可以发现任意形状。

7.4 聚类评估

轮廓系数等指标评估聚类效果。

第8章 分类分析

8.1 决策树

决策树用树形结构进行分类决策。

8.2 随机森林

8.3 逻辑回归

逻辑回归用于二分类问题。

8.4 模型评估指标

准确率、精确率、召回率、F1值。
